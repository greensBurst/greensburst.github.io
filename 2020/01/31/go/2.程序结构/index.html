2020-01-31- 2020-11-22

go-gpl

  1. Go语言中，名称的作用域越大，名称越长且越有意义。

  2. 包级初始化在main函数开始前执行。

  3. 短声明:=左面至少有一个新变量，对旧变量进行赋值操作。

  4. new(T)创建未命名的T类型变量，并返回地址。

  5. var global *int

func f() {                //结束后x还可以访问
    var x int            //x逃逸了
    x = 1                //x被分配在堆空间
    global = &x
}

func g() {                //结束后y被回收了
    y := new(int)        //y被分配在栈空间
    *y = 1
}
