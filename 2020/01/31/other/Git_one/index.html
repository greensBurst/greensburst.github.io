一.Git的安装配置

这里以windows系统为例，如果你的系统是linux，那我默认你已经能够熟练使用或独立安装配置Git工具，如果你的系统是mac，对不起，没有，告辞。

32位系统点这里，64位系统点这里，下载完成后双击打开，你可以自定义Git的安装路径，其余一路默认即可。

安装完成后，打开控制台，输入git version查看版本，若有类似如下提示，即为安装成功。

你还需要在本地配置好你的昵称和用户名。

你可以使用git config user.name,git config user.email查看当前的昵称和邮箱。

二.Git的素质三连

这一步我默认你了解基本的[Dos命令](http://greensburst.github.com/post/dos命令/)，想要快速上手使用Git你需要知道下面的三个命令。

git init <文件夹名>

“init”作initialization初始化的简写，该命令会初始化一个git工作区，若不指定文件夹名，则当前所在目录会被初始化为一个git工作区。

这时控制台会显示你的工作区有修改，在123.txt这个文件中。

git add <文件名>

该命令可以将文件添加到git缓存区，使用git add -A可将当前目录所有改动同步到缓存区，包括添加修改删除操作，git add .会将当前目录的修改和添加同步到缓存区。

git commit -m "本次提交的描述信息"

当你觉得自己的项目到了一定的阶段，就可以使用该命令将你的工作区提交到本地仓库，-m参数即为你对本次提交的描述，这个参数是必须要有的。

git log --oneline

我们继续添加一行内容并保存到我们的本地仓库，使用该命令可以查看我们的历史提交记录，可以看到我们一共做了两次提交，标识名为别为973077d，ba0fca1。

git checkout <标识名>

该命令体现了git工具的强大之处，我们可以使用这条命令随时回退到我们之前提交的版本，可以看到我们执行type 123.txt已经显示内容只剩下一行”111”，已经成功的回退到我们的第一个版本了。
