常对象

2020-03-14- 2020-11-22
-
C++

常对象只能调用常成员函数，无法修改成员变量，mutable除外

class T {
public:
    int a;
    mutable int b;
    void f1(){}
    void f2() const {}
}

const T t;
t.a = 10;   //nok
t.b = 10;   //ok
t.f1();     //nok
t.f2();     //ok
