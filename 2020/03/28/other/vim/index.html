切换编辑模式：

  • i：向光标所在位置前插入
  • I：向光标所在行的行首插入
  • a: 向光标所在位置后插入
  • A：向光标所在行的行尾插入
  • o: 向光标所在位置下一行插入
  • O：向光标所在位置上一行插入
  • s：以删除光标选中字符为前提插入
  • S：以删除光标所在行为前提插入

光标移动：

  • hjkl 左下上右
  • gg 跳转至文件的首行
  • GG 跳转至文件的末行
  • nG 跳转至第n行
  • 0 跳转至光标所在行行首
  • $ 跳转至光标所在行行尾
  • :n回车，末行命令，跳转至第n行

复制粘贴：

  • 单行复制：
    • 光标挪动至待复制行的任意字符上，使用yy复制。移动光标到目标位置，p向下一行粘，P向上一行粘。
  • 多行复制：
    • 光标挪动至待复制多行的首行的任意字符上，使用nyy复制。移动光标到目标位置，p向下一行粘，P向上一行粘。
  • 区域复制：
    1. 将光标挪动至待复制区域的首字符上
    2. 命令模式下按V。左下角出现“可视”提示
    3. 使用hjkl挪移光标，选中待复制区域
    4. 按y复制
    5. 移动光标到目标位置p向后粘，P向前粘

剪切、删除：

  • vim当中的删除，靠剪切实现。
  • 单行剪切、删除：
    • 光标挪动至待剪切行的任意字符上，使用dd剪切。移动光标到目标位置，p向下一行粘，P向上一行粘。不粘即为删除。
  • 删除、剪切多行：
    • 光标挪动至待剪切多行的首行的任意字符上，使用ndd剪切。移动光标到目标位置，p向下一行粘，P向上一行粘。不粘即为删除。
  • 区域剪切：
    1. 将光标挪动至待剪切制区域的首字符上。
    2. 命令模式下，按V。左下角出现 “可视”提示
    3. 使用hjkl挪移光标，选中待剪切区域
    4. 按d剪切
    5. 移动光标到目标位置p向后粘，P向前粘。不粘即为删除
  • 剪切到行尾：
    • d$、D：剪切光标位置到行尾
    • d0：剪切光标位置到行首
    • dw：剪切光标位置到单词的结尾（通常将光标移至单词首字符上）
    • x：删除一个字符。不改变工作模式
    • r：替换一个字符。通常将光标移至待替换字符上，按r，再按新字符。原被选中字符被新字符替换

撤销和反撤销：

  • u 命令模式下，撤销
  • Ctrl+r 命令模式下，反撤销

查找：

  1. 想象一个字符串，去文档中查找
    • 命令模式下，按 “/”，光标出现在末行位置。键入待查找字符串
    • 按 n 查找下一个
  2. 看到一个字符串，查找它后续出现的位置
    • 将光标移至字符串任意字符上
    • 按”*”向后找，按”#”向前找

替换：【末行模式】

  • 单行替换：
    • :s/旧字符串/新字符串，默认只替换光标所在行的一个，想替换全部加上”/g”
  • 通篇替换：
    • :%s/旧字符串/新字符串，默认只替换每行的一个，想替换全部使用加上”/g”
  • 部分替换：
    • :起始行号,终止行号s/旧字符串/新字符串，如果一行有多个，加上”/g”

其他的vim指令：

  • :sp 横屏分 后面可以跟着文件名
  • :vsp 竖屏分 后面可以跟着文件名
  • vim -o file1 file2 上下打开
  • vim -O file1 file2 左右打开
  • ctrl+w+w 切换编辑屏幕
  • :wqall 退出所有
  • :!shell命令 可以在vim中执行shell命令。

vim的配置：

  • 有两处配置位置：
    1. 系统的配置目录位置：/etc/vim/vimrc
    2. 用户的配置：~/.vimrc (注意：是隐藏文件。该文件默认不存在，需要手动创建。)

远程文件传输：

  • scp -r
  • 上传：
    • scp -r 本地文件的访问（相对、绝对） 目标主机用户名@目标主机IP:绝对存储路径（不能使用相对）
  • 下载：
    • scp -r 目标主机用户名@目标主机IP:待下载文件的绝对存储路径（不能使用相对） 本地存储位置（相对、绝对）
