文件描述符表

  • 文件描述符表默认大小1024
  • 每个进程启动后，都有一个文件描述符表
  • 所以每个进程默认能打开1024个文件
  • 前三个文件描述符是默认被使用的
    • stdin 0
    • stdout 1
    • stderr 2
  • 关闭文件会释放文件描述符
  • 每次分配文件描述符选择最小的且未被占用的

Linux系统IO函数

errno

  • 属于linux系统函数库，里的一个全局变量，记录的是错误号

open

  • include
    • sys/types.h
    • sys/stat.h
    • fcntl.h
  • 打开一个已存在的文件
    • int open(const char *pathname, int flags);
    • pathname：要打开的文件的路径
    • flags:对文件的操作权限设置
      • O_RDONLY O_WRONLY O_RDWR 这三个设置是互斥的
    • 返回文件描述符
  • 创建一个新文件
    • int open(const char *pathname, int flags, mode_t mode);
    • pathname：要打开的文件的路径
    • flags:对文件的操作权限设置
      • 必选项：O_RDONLY O_WRONLY O_RDWR 这三个设置是互斥的
      • 可选项：O_CREAT 文件不存在则创建文件
    • mode：八进制的数，创建出的文件权限 = mode & ~umask（$ umask终端查看值）
    • 返回文件描述符

close

  • include <unistd.h>
  • int close(int fd);
    • 成功返回0
    • 失败返回-1，并设置errno

read

  • include <unistd.h>
  • ssize_t read(int fd, void *buf, size_t count); size_t是无符号整数，ssize_t是有符号整数
    • fd：open得到的文件描述符
    • buf：存储读到的数据
    • count：buf大小
    • 返回正数，为实际读到的字节数
    • 返回0，代表文件读完了
    • 失败返回-1

write

  • include <unistd.h>
  • ssize_t write(int fd, const void *buf, size_t count);
    • fd：open得到的文件描述符
    • buf：要写入的数据
    • count：要写入数据的大小
    • 成功返回写入字节数
    • 失败返回-1

lseek

  • include

    • sys/types.h
    • unistd.h

  • off_t lseek(int fd, off_t offset, int whence);

    • fd：open得到的文件描述符
    • offset：偏移量
    • whence：
      • SEEK_SET：设置偏移量为offset
      • SEEK_CUR：设置偏移量为当前位置+offset
      • SEEK_END：设置偏移量为文件大小+offset
    • 跟fseek很像：int fseek(FILE *stream, long offset, int whence);

  • 使用

    • 移动文件指针到文件头：lseek(fd, 0, SEEK_SET);
    • 获取当前文件指针的位置：lseek(fd, 0, SEEK_CUR);
    • 获取文件长度：lseek(fd, 0, SEEK_END);
    • 拓展文件大小100字节：lseek(fd, 100, SEEK_END);
      • 需要进行一次写操作：write(fd, " ", 1);

Linux其他系统函数

stat/lstat

  • include
    • sys/types.h
    • sys/stat.h
    • unistd.h
  • int stat(const char *pathname, struct stat *buf);
    • pathname：操作的文件的路径
    • buf：传出的结构体变量
  • 可以使用$ stat 文件名shell命令查看文件信息
  • stat结构体
  • st_mode
    • 一个十六位整数
    • 可以通过位运算来获取文件的类型和存取的权限

文件属性函数

  • int access(const char *pathname, int mode);
  • int chmod(const char *filename, int mode);
  • int chown(const char *path, uid_t owner, gid_t group);
  • int truncate(const char *path, off_t length);

目录操作函数

  • int rename(const char *oldpath, const char *newpath);
  • int chdir(const char *path);
  • char *getcwd(char *buf, size_t size);
  • int mkdir(const char *pathname, mode_t mode);
  • int rmdir(const char *pathname);

目录遍历函数

  • DIR *opendir(const char *name);
    • name：要打开的目录
  • 返回值：
  • d_type:
  • struct dirent *readdir(DIR *dirp);
    • 返回一个结构体，对应一个文件
  • int closedir(DIR *dirp);
  • 应用案例：获取目录下普通文件个数

dup

  • include <unistd.h>
  • int dup(int oldfd);
  • 返回空闲且最小的文件描述符，这时候返回的描述符指向oldfd指向的文件

dup2

  • include <unistd.h>
  • int dup2(int oldfd, int newfd);
  • 重定向文件描述符
  • 使newfd指向oldfd指向的文件
  • 两个参数相同等于什么也没做
  • 返回值是newfd，失败则返回-1
  • oldfd必须是有效的，newfd是个0-1023的数就行

fcntl

  • include
    • unistd.h
    • fcntl.h
  • int fcntl(int fd, int cmd, ...);
  • 复制文件描述符 === dup
    • int ret = fcntl(fd,F_DUPFD);
  • 设置/获取文件的状态标志
    • 就是open函数的flags参数，其中必选项只读/只写/读写时无法更改的，只能修改可选项
    • 常用可选项
      • O_APPEND 数据追加
      • O_NONBLOCK 设置非阻塞
