nginx是同步的!

2020-04-16- 2020-11-22
-
nginx

nginx通过多进程 + epoll实现，nginx的事件驱动指的就是epoll，而不是通过MQ实现异步的那种

不知道怎么回事一搜nginx都是异步非阻塞，因为这点事研究了好久，压根就不是异步的

master进程里做socket、bind、listen

worker进程里抢accept的处理权，抢到之后就是epoll的一套流程

首先要说的是IO多路复用就是同步非阻塞的，poll就是轮询还异步个p，除非开多线程/多进程，不过要是开了还IO复用干p

nginx的多进程不是在epoll上，epoll的整个处理流程都在worker上，主要为了充分利用多核cpu的优势

真TM气死我了，网上哪来那么多nginx异步非阻塞啊

就算扯了一堆理由往异步上靠，也绝对不是unix的AIO！！！！

POSIX 定义：An asynchronous I/O operation does not cause the requesting process to be blocked;
A synchronous I/O operation causes the requesting process to be blocked until that I/O operation completes;
