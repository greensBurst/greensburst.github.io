网络组成

网络由主机（端系统）、分组交换机、传播介质构成。

网络可划分为核心与边缘（接入网）。

边缘网络通过边缘路由器接入核心网络。

链路层交换机一般位于边缘，实现链路层、物理层；路由器一般位于核心，实现网络层、链路层、物理层。

端系统：PC、工作站、网络服务器、邮件服务器、PDA、游戏终端 …

分组交换机：路由器、链路层交换机 …

传播介质：同轴电缆、铜线、光纤、无线电频谱 …

接入方式

  1. 数字用户线（Digital Subscriber Line）
    • ISP: 本地电话公司的本地中心局（CO）
    • 线路：电话线（双绞铜线）
      • 高速下行信道：50kHz - 1MHz
      • 中速上行信道：4kHz - 50kHz
      • 普通的双向电话信道：0 - 4kHz
    • 端点：
      • DSL调制解调器：数字数据 -> 高频音
      • 数字用户线接入复用器（DSLAM）：模拟信号 -> 数字信号
    • 适用情形：本地中心局5-10英里范围内
  2. 电缆
    • ISP：有线电视公司
    • 线路：混合光纤同轴（Hybrid Fiber Coax, HFC）
      • 光缆：电缆头端（CMTS） -> 地区枢纽
      • 同轴电缆：地区枢纽 -> 各家各户
    • 端点：
      • 电缆调制解调器
      • 电缆调制解调器端接系统（CMTS）：模拟信号 -> 数字信号
    • 特征：共享广播媒体
  3. 光纤到户（Fiber To The Home, FTTH）
    • ISP: 本地电话公司的本地中心局（CO）
    • 方案：
      • 主动光纤网络（AON）：CO直连HOME
      • 被动光纤网络（PON）：增加中间节点
    • 端点：
      • 路由器 + 光纤网络端接器（ONT）
      • 路由器 + 光纤线路端接器（OLT）：光信号 -> 电信号
  4. 卫星、拨号
    • 适用情形：无法提供DSL、电缆、FTTH的地方
    • 卫星：卫星调制解调器、卫星天线
    • 拨号：传统电话线，与DSL模式相同
  5. 以太网、WiFi
    • 通过局域网（LAN）与边缘路由器连接
    • LAN技术
      • 有线LAN：以太网
      • 无线LAN：IEEE 802.11（WiFi）
    • 以太网：双绞铜线、以太网交换机
  6. 广域无线接入：3G、LTE、4G
    • ISP：蜂窝网络提供商

网络核心

存储转发传输：先接收所有的分组到交换机队列，接着处理队列中的分组，每处理成功一个分组，转发到输出链路。

分组丢失（丢包）：输出队列已满，交换机会丢弃新到的分组。

转发表：将目的地址的一部分映射为输出链路，转发表由路由选择协议自动设置。

电路交换：相比于分组交换，预留了端系统间通信会话期间所需要的资源。电路交换的实现通过频分复用（FDM）或时分复用（TDM）。因特网属于分组交换网络。

分组交换网

处理时延：检查分组首部、转发决议 …

排队时延：排队时延：输出链路正在传输其他分组。

传输时延：交换机转发的时间，不包括首分组和末分组到达之间的时间。

传播时延：交换机之前传播需要的时间。

吞吐量：瓶颈链路的传输速率，如今因特网吞吐量的限制主要在接入网。

网络协议

应用层（HTTP、SMTP、FTP）：应用程序间分组（报文）的格式。

运输层（TCP、UDP）：应用程序间传送分组（报文段）的标准。

网络层（IP）：主机间传送分组（数据报）的标准。

链路层（DOCSIS）：网络元素间传送分组（帧）的标准。

物理层：物理节点间传送比特的标准，不同物理介质有不同的标准。
